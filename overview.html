Online Examination System

Now-a-days, Online examination system has become popular for competitive examinations because of its unique features such as auto-evaluation, speed and accuracy. In such a system, students are asked to select answers from multiple options given for a single question. Likewise, there are several questions which appear in the students’ systems. The questions and multiple options are saved in a database along with desired answers. Typically, a student can edit an answer after saving it, however, editing cannot be done after submitting the answer. Another user is also there – administrator. The administrator can create, modify and delete questions and accordingly, the question is updated in the website.

Role of Administrator

  • The administrator will have a unique email and password.
  • He/she should login with those credentials
  • The administrator can now Create, Modify or Delete questions.
  • Accordingly the questions will be updated in the website.

Role of Student

  • A first time user should signup by providing all the details like Username and Password.
  • An Existing User should signin by providing Email and Password.
  • The student can now access the tests and can start the exam.
  • The student should save the answers and submit the exam at the end.

After Completion of exam..

  • Result of the student will be displayed.
  • Display the result in descending order according to obtained marks with Roll Number.
  • Some other analytics of all students will be displayed.
